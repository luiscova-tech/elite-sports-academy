# Purpose

This document defines how players progress through Elite Sports Academy.

Progression should introduce new mechanics gradually while giving players meaningful goals.

---

# Core Philosophy

The player should never feel overwhelmed.

Each campus introduces one or two major gameplay concepts before the next campus unlocks.

Progression should reward mastery, not grinding.

# Unlock Philosophy

Every new campus should introduce:

- One new gameplay mechanic
- One new strategic decision
- One new long-term objective

Never introduce more than two major systems at once.

---
