# Headquarters Buildings

All Headquarters buildings use Academy Funding instead of Training Points.

Default Max Level: 20

---

# Design Rules

- Campus buildings use Training Points.
- Headquarters buildings use Academy Funding.
- Upgrade costs should approximately double each level.
- Max levels may increase in future expansions.
