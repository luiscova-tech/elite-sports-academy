# Purpose

This document defines the balance philosophy for Elite Sports Academy.

Exact numbers may change during playtesting, but the progression principles should remain consistent.

---

# Core Philosophy

The game should reward smart planning over excessive grinding.

Every upgrade should feel meaningful.

Every new campus should noticeably improve academy growth.

---

# Resource Scaling

## Training Points

Generated locally.

Cannot be transferred between campuses.

Primarily used for:

- Facilities
- Coaches
- Local upgrades

---

## Academy Funding

Generated globally.

Comes from:

- Campuses
- Competitions
- Sponsors

Used for:

- Headquarters
- Global upgrades

---

## Reputation

Global progression metric.

Used to unlock:

- Campuses
- Sponsors
- Championships
- Prestige content

---

# Upgrade Philosophy

Each upgrade should provide a noticeable benefit.

Avoid upgrades below 5%.

Avoid mandatory upgrades.

Players should have multiple viable upgrade paths.

---

# Offline Progression

Limited by Operations Capacity.

Offline rewards should never exceed the current cap.

Offline progression should encourage returning regularly without punishing longer breaks.

---

# Difficulty Curve

Early Game

Fast progression.

Frequent rewards.

---

Mid Game

Requires planning.

Introduces tradeoffs.

---

Late Game

Focuses on optimization.

Multiple systems interact.

---

# Economy Rules

Training Points remain local.

Academy Funding remains global.

No premium currency should be required for progression.

---

# Future Balancing

Future updates may adjust:

- Upgrade costs
- Reward values
- Competition payouts
- Sponsor rewards
- XP curves

These adjustments should preserve player progress whenever possible.

---

# Design Philosophy

A successful player should feel rewarded for making good strategic decisions, not simply spending more time grinding.
