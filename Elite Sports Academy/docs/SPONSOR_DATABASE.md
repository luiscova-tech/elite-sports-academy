# Purpose

Defines sponsor tiers, bonuses, and unlock requirements.

Sponsors provide global benefits without becoming mandatory.

# Sponsor Rules

- Only a limited number of sponsors may be active simultaneously.
- Better sponsors require maintaining reputation.
- Sponsors may offer special event objectives.
- Some sponsors are sport-specific.
