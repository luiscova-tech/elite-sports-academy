# Navigation Rules

- Every major screen should be reachable within one tap.
- Resource totals remain visible at all times.
- Navigation should remain consistent across desktop and mobile.

---

# Design Philosophy

Players should always know where they are and where to go next.

Navigation should support quick gameplay with minimal unnecessary taps.
