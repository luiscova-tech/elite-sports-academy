# Purpose

Defines upgrade levels shared throughout the game.

Individual buildings may override values, but this serves as the default progression.

---

# Design Rules

- Costs should increase faster than rewards.
- Early upgrades should feel inexpensive.
- Late-game upgrades should require long-term planning.
- No upgrade should feel mandatory.
